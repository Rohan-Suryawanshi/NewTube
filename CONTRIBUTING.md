# Contributing Guidelines

## Getting Started

1. Fork the repo
2. Clone your fork
3. Run `npm install` or `bun install`

## Code Style

- Use ESLint + Prettier
- Write TypeScript types

## Commit Messages

- Use conventional commits: `feat:`, `fix:`, `chore:`, etc.

## Pull Requests

- Run tests before creating a PR
- Describe what the PR does clearly
